Lifeline is a **web and mobile platform** that streamlines volunteer coordination during disasters like floods, fires, and typhoons.  
It allows volunteers to register, log emergencies, and receive tasks, while coordinators can track deployments **in real-time**.  

By integrating **blockchain** for immutable records and **AI** for route optimization, Lifeline ensures faster, organized, and accountable emergency response.
